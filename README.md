This course is a graduate introduction to the inner workings, training, and evaluation of Large Language Models.  It is offered at MSOE under the course code [CSC 6714](catalog.msoe.edu/preview_course.php?catoid=45&coid=47525).

For more information about the course, please contact Josiah Yoder at MSOE.

See the [syllabus](syllabus/syllabus.md) and [weekly schedule](syllabus/weekly-schedule.md) to get started. Everything else is linked from there.

These course materials are best viewed in Obsidian. They are converted to .html with Pandoc to publish on Canvas. Please contact the author (Josiah Yoder at MSOE) for more details on how to publish the materials in this course to a Canvas course.
